## 🗂 功能模块详解

### 🧭 战略立项 (Inception)
从零开始规划研究方向，AI 协助分析研究前沿、生成选题建议，帮你找到有价值的科研切入点。

### 📡 行业动态 (Industry Trends)
实时追踪学科领域最新研究动态和行业趋势，不错过任何重要发现。

### 📋 研究看板 (Dashboard)
科研项目的指挥中心——统计总览、进度跟踪、待办事项和关键指标一目了然。

### 🧪 课题中心 (Project Hub)
- **课题管理**：创建和管理多个科研课题
- **里程碑规划**：设定关键节点和截止时间
- **周计划**：每周实验安排和进度追踪
- **实验日志**：自动记录和整理实验过程
- **团队协作**：邀请成员、分配任务、实时在线协同

### 👥 团队矩阵 (Team Matrix)
科研团队管理平台，可视化团队成员分工、协作关系和工作负载。

### 🧠 中心大脑 (Research Brain)
知识图谱系统，自动构建研究课题之间的关联网络，发现隐藏的交叉研究灵感。

### 📚 情报档案 (Literature Archive)
- **文献管理**：批量导入、分类管理科研文献
- **AI 摘要提取**：自动从 PDF 提取关键信息
- **多分类标签**：灵活的标签和分类系统
- **文献笔记**：为每篇文献添加阅读笔记和批注

### ⚛️ 机理推演 (Mechanism)
- **反应机理可视化**：拖拽式构建化学反应路径图
- **双金属协同机理图**：专业级催化机理示意图
- **AI 辅助推演**：基于文献数据智能推测反应机理

### 🔬 实验表征中心 (Characterization Hub)
- **XRD 分析**：衍射图谱解析、物相检索 (COD数据库)、晶体结构鉴定
- **XPS 分析**：结合能分析、化学态鉴定、元素定量
- **SEM 分析**：微观形貌分析、AI 颗粒检测、孔隙率计算
- **TEM 分析**：高分辨像分析、晶格间距测量
- **一键 Benchmark**：自动对标文献数据，生成对比报告

### 📦 库存管理 (Inventory)
实验室耗材、试剂和设备管理系统，跟踪库存状态，设置低库存预警。

### 🧩 DOE 迭代 (Design of Experiments)
实验设计优化工具——正交实验、响应面分析，AI 辅助分析实验结果并推荐下一步实验参数。

### 🔀 实验路线 (Flowchart)
可视化实验操作流程图生成器，自动将实验步骤转化为清晰的流程图。

### 📈 数据分析 (Data Analysis)
- **数据导入**：支持 CSV/Excel/TXT 等多种格式
- **统计分析**：描述性统计、相关性分析、假设检验
- **视觉分析**：AI 驱动的图像识别与分析
- **交互式图表**：可拖拽、可缩放的动态数据可视化

### 🎨 科研绘图中心 (Figure Center)
- **图表拼装**：多子图自由排列，生成期刊级组合图
- **桑基图设计器**：学术级桑基图，自定义节点和流量
- **弦图生成器**：高精度圆形弦图，展示多元素关联
- **时间线编辑器**：研究进展时间线可视化
- **模板库**：预设多种期刊图表模板
- **高分辨率导出**：支持 SVG/PNG/PDF，最高 1200 DPI

### 🎬 视频工坊 (Video Lab)
科研成果展示视频制作工具，帮助制作学术报告和成果汇报的可视化内容。

### ✍️ 写作工坊 (Writing Studio)
- **学术编辑器**：双栏所见即所得编辑，首行缩进、段落排版
- **期刊预览**：实时预览论文最终排版效果（多栏分页）
- **LaTeX 公式**：行内/行间公式渲染
- **文献引用**：自动管理引用（Nature/Science/IEEE/APA/JACS 格式）
- **AI 润色**：三档润色力度（轻度/中度/深度），保持学术语言规范
- **FigRef 交叉引用**：自动编号和引用图表（Fig. 1, Table 2...）

### 🎙️ 语音伴侣 (Voice Companion)
实验操作时解放双手——用语音指令控制应用、记录实验笔记。

### 🤖 AI 智能助手 (AI CLI)
全局 AI 对话窗口，随时提问科研相关问题，获取数据分析建议、写作润色、代码生成等帮助。
